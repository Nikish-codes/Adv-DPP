## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Nikish-codes/Adv-DPP.git
   ```

2. Install dependencies:
   ```bash
   cd Adv-DPP
   npm install
   ```

3. Start the development server:
   ```bash
   npm run start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
